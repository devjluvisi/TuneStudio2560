Detailed Description

The class for managing the current song being played. This state also uses the bottom row of the LCD to display different controls on a varying interval.

Author
Jacob LuVisi

Note: The way that songs are played back is different then usual. This class does NOT use the song.cpp way of playing back songs and instead it relies on a global index and the general loop() function instead of a seperate while loop.
