\hypertarget{main_8cpp_A}{}\doxysubparagraph{Brief Description}\label{main_8cpp_A}
The prgm\+State variable represents the current \mbox{\hyperlink{class_program_state}{Program\+State}} the program is in on a global scale. ~\newline
 The prgm\+State is a pointer to a larger \mbox{\hyperlink{class_program_state}{Program\+State}} object which was allocated on the heap.~\newline
 Whenever we need to interact with the current user state we access the prgm\+State variable which is only scoped to the main class.~\newline
 To change the prgm\+State from another class you either need to use an interrupt from a supporting \mbox{\hyperlink{class_program_state}{Program\+State}} or use the update\+\_\+state method.~\newline
 The prgm\+State variable {\bfseries{SHOULD NEVER}} be allocated with new() outside of update\+\_\+state and the \mbox{\hyperlink{main_8cpp_a4fc01d736fe50cf5b977f755b675f11d}{setup()}} method. ~\newline
 To learn more about program states view the \mbox{\hyperlink{state_8h}{state.\+h}} file.
