MakerStudio2560 is a song creation, editing, and playback tool made to assist the TuneStudio2560 for users who wish to create their own songs outside of TuneStudio2560.
MakerStudio2560 was created to allow the fast creation of songs outside of TuneStudio2560.
The benefits of MakerStudio2560 include an interactive Web GUI when creating songs which allows the creation of complex songs in a faster manner then what is possible on TuneStudio2560.
Although TuneStudio2560 provides a user friendly way in creating songs, MakerStudio2560 provides more functionallity with a keyboard and cursor control scheme.
Of course, MakerStudio2560 is completely backwards compatible with TuneStudio2560 in its handling of Song files, so any songs you create in TuneStudio2560 will work here and any songs you make here will work in TuneStudio2560.
Start by selecting a song file already on your SD card or creating a new song file now.

After you have uploaded or created a new song file you first should adjust the attributes which are above the Song Data text.
These attributes represent keys for both TuneStudio2560 and MakerStudio2560 which are used to playback your song and recognize it.
MakerStudio2560 has custom attributes such as Author, Creation Date, and Last Edit Date. These are not read by TuneStudio2560.
When you have adjusted your attributes, add or remove notes by clicking on the individual buttons.
Note that MakerStudio2560 allows you to have a song as small as one in length but TuneStudio2560 requires at least 8 notes to play.
There is also no set limit on notes in MakerStudio2560, so make sure the number of notes you add is less then your Arduino's maximum allowed setting. (255 default)
When you have finished adding notes, you can playback your song using the media controls as well as adjust the volume of the song playback.
There is also a variety of other UI settings to change like font size, font type, dark mode.
Your progress is automatically saved every 5 seconds by default. Adjust it by clicking on the dropdown with the green text.
When finished, click on Download button to download a song file or Copy to copy the text of the song file.
MakerStudio2560 and TuneStudio2560 communicate using the same file format so songs between them can be interchanged at will.

Your progress on MakerStudio2560 is automatically saved.
The interval for which your progress is saved can be adjusted by clicking on the dropdown with green text.
AutoSave can be enabled or disabled by clicking on a checkbox.
The interval can be adjusted by changing the value of the interval text box.
When finished you must click Save Settings or else your auto save settings will not save.

There is a variety of UI customization which users can do on MakerStudio2560.

Toggle Dark Theme: Toggle the dark/light theme by clicking on the Toggle Dark Mode hyperlink.
Your dark mode settings are automatically saved.
Change Font Family: Users can adjust their font to be one of five pre-selected fonts by clicking on the Change Font hyperlink.
The changes to the font family are automatically saved.
Change Font Size: The font size for the song data table is adjustable by clicking on the + or - buttons.
Changes to the font size will not be saved and only last for the duration that the website instance is running.

Each song which is created has individual song attributes which can be adjusted.
Song Name: An attribute read by the Arduino. Tells the name of the song file. Must be between 1-8 characters.
Song Size: A non-adjustable attribute tells the length of the song excluding the END note.
Author: An attribute NOT read by the Arduino. Tells the author of the song and is saved into the song file text/data. Attribute can be read by MakerStudio2560 for future use.
Tone Length: An attribute read by the Arduino. Adjust the length that each individual tone in the song plays for (in milliseconds). Numbers must be above 0 and below 255.
Tone Delay: An attribute read by the Arduino. Adjust the delay between each of the individual tones in the song.
Creation Date: An attribute NOT read by the Arduino. Describes the creation date of the song file in MakerStudio2560. Can be read by MakerStudio2560.
Last Edit Date: An attribute NOT read by the Arduino. Describes the last edit date of the song file in MakerStudio2560. Can be read by MakerStudio2560.

Editing song data works somewhat similar to that of TuneStudio2560.
MakerStudio2560 starts with a default B0 note for new songs.
Additional notes can be added by clicking on the button.
When a new note is added, its default value is set the same as the note it was just added from.
Notes can be deleted by clicking on the button.
There must be at least one note in the song excluding the END note.
After a note has been added, its pitch can be adjusted by clicking on the dropdown.
When a new pitch is selected, the frequency column automatically updates with the new frequency (in hz) of that note.
MakerStudio2560 is different then TuneStudio2560 in that notes can be changed at any point in the song and notes can be inserted/removed at any point in the track.

Similar to on TuneStudio2560, songs can be played back while editing.
The box below the Song Data is the media control box and controls song playback.
A text header at the top describes the current state of the playback.
The progress bar and time describe the current progress of the track in real life seconds.
The fast forward/rewind buttons allow a user to skip 10% ahead or 10% behind in the songs track.
The volume slider allows the volume of the song to be adjusted during playback.
The Play/Pause Button allows a song to be started or paused at any point.
The Restart Button allows a song to be reset and played again when finished.
The Stop Song Button immediately stops the song at any point and resets progress.
The track that the media control area reads is updated every time an auto-save is run.

When satisfied with a song, an author can choose to download the song or copy its contents.
The Download button will download (client-side) a .txt file with the name being the name of the song.
This new song file is automatically prepared to run on the Arduino.
Simply drag the file onto the Arduino SD card and run it!
The Copy button will copy all of the data to the clipboard instead of downloading a file. This data can then be pasted anywhere.
